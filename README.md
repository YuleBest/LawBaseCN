# LawBaseCN

LawBaseCN 是一个中国法律法规数据库，旨在收集中国现行所有法律法规条文并进行展示

在线浏览：[lawbase.cn.yule.ink](https://lawbase.cn.yule.ink/)

## 为什么诞生

因为作者有时候需要在网上搜索一些法律条文，而百度等搜索引擎搜出来很多推广的内容，官方的国家法律法规数据库用起来不方便，故此项目诞生。

## 目标是什么

目标是建成一个由社区驱动的，收录中国所有现行法律法规条文的数据库，并支持多端浏览、条目搜索、筛选、分类等。

## 项目进度

目前项目处于启动阶段，仅有本人一人进行开发和收录工作，需要更多志同道合的伙伴加入！

目前已经收录了[一些法律](https://lawbase.cn.yule.ink/law/)，可以作为后续扩展的参考。

## 如何成为贡献者

1. Fork 本仓库

2. 您可以参考现有条目，新增或修改内容

3. 在本仓库发起 PR，我们会及时处理您的修改

感谢您的贡献，一起让这个项目变得更好！
